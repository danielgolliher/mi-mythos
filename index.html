MI Mythos

Source-grounded policy briefs on New York City and New York State, on demand.

New York governance runs on three layers: legislation, agency rule-making, and case law. Most coverage skips two of them. MI Mythos generates a research-grade prompt that pulls from all three — primary bill text, hearing transcripts, comptroller audits, NYCRR/RCNY rules, and active litigation — and renders the result as a printable, footnote-linked HTML brief.

How to use this

  
    
    
  1. Type a NYC or NYS policy question into the box above — a bill ID (Int 0704-2022), a law (LL 64/2023), a topic ("source-of-income discrimination enforcement"), or a court case (Vincent v. Adams) all work.
    2. 
    
  2. Click Generate brief prompt. The page produces a self-contained research prompt that embodies the methodology used to build the example brief linked below.
    3. 
    
  3. Click Copy to clipboard and paste into Claude Code (terminal) or Claude.ai. Claude will pull from the cipher legislative graph, official Senate / Assembly pages, RCNY / NYCRR rule-makings, NYS Comptroller audits, IBO reports, and case law — then assemble the brief into a single durable HTML file at ~/projects/<slug>/index.html with all source documents saved alongside.
    4. 
    
  4. The brief includes a floating TOC, a print button, hover-tooltip footnotes for every named person, expandable verbatim transcript blocks, and an administrative-and-case-law section.
    5. 
  

  

    If you have Claude Code installed locally, you can also invoke the underlying skill directly with /ny-policy-brief <your topic>.
